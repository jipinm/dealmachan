Application Architecture

1. Admin Application

Server-side rendered PHP application for administrative users with different access levels.

PHP 8.1+ MySQL HTML5 CSS3 Bootstrap 5 jQuery JavaScript

Note: This is a traditional PHP application with direct MySQL database access. Pages are rendered server-side with Bootstrap for UI and jQuery for client-side interactions.

2. API Application

RESTful API backend serving both public and authenticated endpoints for customer and merchant applications.

PHP 8.1+ MySQL REST API JWT Auth JSON

Features: Token-based authentication, standardized response format, versioned endpoints, comprehensive error handling.

3. Merchant Application

Modern single-page application for merchant account management and operations.

Vite React 18+ TypeScript CSS Modules JWT Auth

Backend: Consumes PHP REST APIs for all data operations.

4. Customer Application

Customer-facing single-page application for exploring offers, redeeming coupons, and engaging with the platform.

Vite React 18+ TypeScript CSS Modules JWT Auth

Backend: Consumes PHP REST APIs for all data operations.

System Overview

Deal Machan is a comprehensive discount and coupon management platform connecting customers, merchants, and administrators. The system facilitates offer exploration, coupon redemption, merchant management, and customer engagement through mystery shopping, surveys, and referral programs.

User Roles & Access Levels

Customer Types

Standard Customer Premium Customer Deal Maker

Customers can register through multiple channels: self-registration, merchant app, admin registration, preprinted cards, or automated merchant profile generation.

Merchant Account

Business Owner Store Manager

Merchants manage multiple store accounts with capabilities for coupon generation, customer analytics, and profile management.

Admin Hierarchy

Super Admin City Admin Sales Admin Promoter Admin Partner Admin Club Admin

Partner Admin and Club Admin have limited access to manage customers under them with prefixed customer caps set by Super/City Admin.

Customer Account Functionalities

Core Features

  • Explore, subscribe, and redeem offers and discounts
  • Receive, accept, and redeem gift coupons from admin
  • Receive and redeem store coupons from merchants
  • Profile management and customization
  • Subscription management (yearly basis with renewal)

Engagement Features

  • Participate in mystery shopping programs
  • Complete surveys and questionnaires
  • Get assistance from deal makers
  • Explore, rate, and review merchants
  • Refer customers to Deal Machan

Merchant Account Functionalities

Account Management

  • Multi-store account management
  • Profile management and customization
  • Acquire label branding for credibility
  • Request account upgrades
  • Message communication with admin
  • Subscription renewal and upgradation

Customer Interaction

  • Generate and redeem coupons
  • Generate customer accounts
  • Access customer analytics from sales registry
  • Generate, gift, and redeem store coupons
  • Receive and respond to customer grievances
  • Display offers, discounts, gallery, and contact info
  • Manage ratings and reviews

Admin Backend Management

Customer Management

  • List, add, and manage customers
  • Block/unblock/upgrade customer accounts
  • Filter customers by various criteria
  • Bulk select customers and gift coupons (with or without acceptance)

Deal Maker Management

  • Add deal makers by selecting customers and appointing them
  • Receive requests from customers to become deal makers
  • Review and approve deal maker applications
  • Task monitoring and performance tracking

Merchant Management

  • List, add, and manage merchants
  • Block/unblock/upgrade merchant accounts
  • Filter merchants by various parameters
  • Designate premium partners and assign labels
  • Review business enquiries for registration

Priority Listing: Merchants and coupons are listed based on premium partner status, label assignment, and newest to oldest.

Card Management System

  • Add cards with variant selection and parameter configuration
  • Upload card images and generate card numbers dynamically
  • Block/unblock cards for new generations
  • Generate preprinted cards in bulk
  • Assign preprinted cards to merchants or admins before customer allocation

Coupon Management

  • Add and list coupons with review and approval workflow
  • Filter coupons by multiple criteria
  • Block and unblock coupons
  • Manage store coupons: bulk approval and live coupon display

Dynamic Content Management

  • Labels: Authenticate merchant credibility
  • Tags: Special filters for categories and subcategories
  • Master Data: City, Area, Location, Profession, Job Title, Day Type

Special Modules

  • Survey Master: Create and manage customer surveys
  • Mystery Shopping: Coordinate mystery shopping activities
  • Contest & Winners: Manage contests and announce winners

Technical Architecture Details

Admin Application (PHP Traditional)

  • Architecture: MVC pattern with server-side rendering
  • Database Access: Direct MySQL queries using PDO with prepared statements
  • Session Management: PHP sessions for authentication and user state
  • Security: CSRF tokens, input sanitization, SQL injection prevention
  • UI Framework: Bootstrap 5 for responsive design
  • JavaScript Libraries: jQuery for AJAX, DataTables, Chart.js, Select2
  • File Structure: Separate folders for controllers, models, views, includes
  • Configuration: config.php for database and environment settings

API Application (PHP REST)

  • Framework: Slim Framework or Laravel for API development
  • Architecture: RESTful design with service layer pattern
  • JWT Library: Firebase PHP-JWT for token generation and validation
  • Database Layer: PDO with prepared statements
  • Middleware: Authentication, CORS, rate limiting, request validation
  • Response Format: Standardized JSON responses
  • Error Handling: Centralized exception handling with proper HTTP status codes
  • API Versioning: URL-based versioning (/api/v1/)

React Applications (Customer & Merchant)

  • Build Tool: Vite for fast development and optimized builds
  • Language: TypeScript for type safety
  • State Management: React Context API + useReducer or Zustand
  • Routing: React Router v6 with protected routes
  • API Communication: Axios with interceptors for JWT management
  • Form Handling: React Hook Form with Zod validation
  • Styling: CSS Modules for scoped styling
  • Charts: Recharts or Chart.js for analytics
  • Notifications: React Toastify or custom toast system
  • Image Handling: Lazy loading, compression, CDN integration

Database (MySQL)

  • Indexing: Primary keys, foreign keys, composite indexes on frequently queried columns
  • Relationships: Proper foreign key constraints with ON DELETE/UPDATE rules
  • Normalization: 3NF for most tables, denormalization where needed for performance
  • Full-Text Search: For merchant and coupon search functionality
  • Triggers: For audit logging and automated status updates
  • Stored Procedures: For complex operations like bulk gifting
  • Backup Strategy: Daily automated backups with point-in-time recovery

Security & Performance

Security Measures

  • SQL injection prevention with prepared statements
  • XSS protection with input sanitization and output encoding
  • CSRF tokens on all forms (Admin App) and API requests
  • Rate limiting on API endpoints
  • File upload validation (type, size, content)
  • Role-based access control (RBAC)
  • JWT token authentication for React apps
  • Session-based authentication for Admin app
  • Audit logging for sensitive operations
  • HTTPS-only in production

Performance Optimization

  • Database query optimization with EXPLAIN analysis
  • Redis caching for frequently accessed data
  • CDN for static assets and images
  • Lazy loading for images and React components
  • Code splitting and bundle optimization in Vite
  • API response caching with ETags
  • Database connection pooling
  • Gzip compression for API responses
  • Pagination for large datasets
  • Minification of CSS/JS in production

Deployment & DevOps

Development Environment

  • Admin App: PHP 8.1+, Apache/Nginx, MySQL 8.0+
  • API App: PHP 8.1+, Composer, Apache/Nginx
  • React Apps: Node.js 18+, npm/yarn, Vite
  • Version Control: Git with GitFlow workflow
  • Local Development: XAMPP/WAMP for PHP apps, Vite dev server for React

Production Environment

  • Hosting: AWS, DigitalOcean, or VPS
  • Web Server: Nginx with PHP-FPM for PHP applications
  • React Apps: Static build files served by Nginx or CDN
  • Database: MySQL with read replicas for scalability
  • Caching: Redis for session and data caching
  • SSL: Let's Encrypt with auto-renewal
  • Backup: Automated daily backups with 30-day retention
